Commercial

  • School Information System Integration

    NextFrontier Technologies Inc. designed and implemented GleeWorks Message Broker, an integration between:

    • GleeWorks J2EE based electronic forms, eHealth Records and form tracking system and
    • WhippleHill .NET based school information system

    The Message Broker provides two-way data synchronization between GleeWorks and WhippleHill that offers schools time and cost saving features for enabling the parents to quickly update and submit the necessary online forms with the most current and accurate information.

    Built upon SOA architecture, the Message Broker handles complex data mapping and service orchestration between two very different systems. It also comes with an HTML5/JavaScript based Administration console, batch import/export tool, daily backup and email alert features that significantly reduces the time and resource for maintenance.
